# 💻 PC Mania - Trabalho de POO usando Java

Repositório destinado ao projeto "PC Mania", desenvolvido como parte de atividades práticas e avaliações disciplinares envolvendo a linguagem Java e os paradigmas da Programação Orientada a Objetos.

---

## 🎯 Sobre o Projeto
**PC Mania** é um sistema de simulação de compra e montagem de computadores (PCs). O usuário/cliente pode escolher configurações diferentes de máquinas com variados componentes, e o sistema gerencia as configurações, modelos e o cálculo de preços finais, aplicando conceitos de POO na prática.

### ⚙️ Requisitos
- **Java JDK 8+** instalado.
- Terminal ou IDE (Visual Studio Code, Eclipse, IntelliJ, etc) configurada rodar aplicações console.

---

## 🚀 Como Executar

1. Clone este repositório para o seu computador:
```bash
git clone https://github.com/MaskDMoa/Trabalho-de-POO-usando-java-.git
```
2. Navegue até a pasta de fontes `PC_mania` usando o terminal.
3. Compile os arquivos e rode a classe principal (`Main`):
```bash
javac *.java
java Main
```

## 🧩 Pilares de POO Aplicados
- **Abstração:** Classes modelando PCs reais (Componentes, Memória, Hardware).
- **Encapsulamento:** Agrupamento de dados e modificadores de acesso (private/public).
- **Interação entre Objetos:** Associações e composições das peças aos modelos (PCs).
